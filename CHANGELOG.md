# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2024-12-22
### Added
- **Rebranding:** Project renamed to **Lito** (formerly Vietnamese TTS).
- **Multi-language Support:** Added support for English (US) and Chinese (Mandarin) voices.
- **Simplified UI:** Curated list of high-quality voices (1 Male, 1 Female per language) for a cleaner experience.

## [1.0.0] - 2024-12-22
### Added
- Initial release of the Vietnamese TTS Desktop Application.
- Support for text input and file upload (.txt, .md, .pdf).
- Integration with Microsoft Edge TTS (online voices).
- MP3 audio generation and playback.
- Basic GUI using Tkinter.

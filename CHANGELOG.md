# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-22
### Added
- Initial release of the Vietnamese TTS Desktop Application.
- Support for text input and file upload (.txt, .md, .pdf).
- Integration with Microsoft Edge TTS (online voices).
- MP3 audio generation and playback.
- Basic GUI using Tkinter.
